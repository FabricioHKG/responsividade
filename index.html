Exemplo Respansividade

  • ---- As imagens estão com 100% de width (largura) para facilitar a respansividade/diminuição da mídia com a pagína.
  • --- Vale dizer que as imagens pequenas são acionadas quando a página chega a 600px de width (largura). Lembrando que a imagem pequena esta dentro de uma tag source:media

    • EX:
    source media="(min-width: )
    Na parte de: media="(min-width: ) mudei para: media="(max-width: 600px)

                    
                    
     <picture> 
    
                        <source media="(max-width: 600px)" srcset="imagens/irina-blok-pq.jpg"> 
    
                        <img src="imagens/irina-blok.jpg" alt="Criadora do mascote android"> 
    
                    </picture>
